Tutorial

1 - Enter in the Enterprise Manager of the dev ambient

2 - Search the service in the "deployed composites" tab

3 - In the service page click in the symbol

4 - Copy the WSDL url

5 - Open the SOAP UI program and create a new project by clicking in the SOAP button. Paste the url of the service in the "Initial WSDL" input form. Change the project name as desired.

6 - Save the project in a folder and that will create the xml

7 - Open the generator and choose the saved xml file. Then click in the "Extract info from file" button. This action will extract the Service Name, the Service Type and the requests info automatically.

8 - Complete the other fields in the form. Change these fields as desired. You can consult how the IC is going bellow.

9 - Click in the Copy buttons and paste them in the creation of new iC
