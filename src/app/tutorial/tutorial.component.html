Tutorial

Prerequisites

1 - Access to dev environment.

2 - SOAPUI aplication.

Steps

1 - Enter in the Enterprise Manager of the dev environment.

2 - Search the service in the "deployed composites" tab and select it.

3 - In the service page click in the symbol .

4 - Copy the WSDL url.

5 - Open the SOAP UI program and create a new project by clicking in the SOAP button. Paste the url of the service in the "Initial WSDL" input form. Change the project name as desired.

6 - Save the project in a folder. That will create the xml project file that you will use in the IC generator app.

7 - Open the Ic generator app and choose the saved xml file. Then click in the "Extract info from file" button. This action will extract the Service Name, the Service Type and other info that will automatically be put in the IC.

8 - Complete the other fields in the form. Change these fields as desired. You can check how the IC structure is going to be bellow.

9 - Click in the Copy buttons and paste them in the creation of new iC in JIRA.
